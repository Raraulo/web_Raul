# 1. Filtrar outliers extremos para mejorar la visualización
# Filtramos vuelos con distancias menores a 4000 millas y retrasos menores a 300 min (5 horas)
# Esto elimina casos atípicos que deforman la escala del gráfico.
df_filtered = df_delayed.filter((df_delayed["ARRIVAL_DELAY"] < 300) &
                                (df_delayed["DISTANCE"] < 4000))

# 2. Muestreo (Sampling)
# Graficar 2 millones de puntos es ineficiente. Tomamos una muestra aleatoria del 1%.
# El parámetro 0.01 representa el 1%, seed=42 asegura que el resultado sea reproducible.
df_sample = df_filtered.select("DISTANCE", "ARRIVAL_DELAY").sample(False, 0.01, seed=42).toPandas()

# 3. Visualización con Scatter Plot
plt.figure(figsize=(11, 6))

# Usamos 'regplot' en lugar de 'scatterplot' para ver la línea de tendencia (opcional)
# 'scatter_kws' ajusta la transparencia (alpha) para ver mejor la densidad de puntos.
sns.regplot(
    data=df_sample,
    x="DISTANCE",
    y="ARRIVAL_DELAY",
    scatter_kws={'alpha':0.3, 's':10},
    line_kws={'color':'red'},
    color="#2c3e50"
)

# Configuraciones del gráfico
plt.title("Relación entre Distancia del Vuelo y Retraso en la Llegada", fontsize=15)

3. ¿Qué aerolínea tiene el mayor porcentaje de vuelos con retrasos?



* Calcula el total de vuelos por aerolínea y cuántos de ellos tuvieron retrasos.
* Visualiza el porcentaje de retrasos por aerolínea en un gráfico de barras.
In [13]:
# 1. Calcular el total de vuelos por aerolínea (Dataset limpio completo)
total_vuelos = df_clean.groupBy("AIRLINE").count().withColumnRenamed("count", "total_vuelos")

# 2. Calcular el total de vuelos retrasados por aerolínea
retrasos_vuelos = df_delayed.groupBy("AIRLINE").count().withColumnRenamed("count", "vuelos_retrasados")

# 3. Unir (Join) ambos resultados y calcular el porcentaje
# Usamos un 'inner' join porque queremos aerolíneas presentes en ambos conteos
stats_aerolineas = total_vuelos.join(retrasos_vuelos, "AIRLINE")

# Calculamos el porcentaje: (retrasados / total) * 100
stats_aerolineas = stats_aerolineas.withColumn(
    "porcentaje_retraso",
    (stats_aerolineas["vuelos_retrasados"] / stats_aerolineas["total_vuelos"]) * 100
)

# 4. Convertir a Pandas y ordenar para la gráfica
stats_pd = stats_aerolineas.orderBy("porcentaje_retraso", ascending=False).toPandas()

# 5. Visualización
plt.figure(figsize=(12, 6))

sns.barplot(
    data=stats_pd,
    x="AIRLINE",
    y="porcentaje_retraso",
    hue="AIRLINE",
    palette="coolwarm",
    legend=False
)

# Añadir etiquetas de texto sobre las barras para mayor precisión
for i, row in stats_pd.iterrows():
    plt.text(i, row.porcentaje_retraso + 0.5, f'{row.porcentaje_retraso:.1f}%',
             ha='center', va='bottom', fontsize=9)

# Configuraciones finales
plt.title("Porcentaje de Vuelos con Retraso por Aerolínea", fontsize=15)
plt.ylabel("Porcentaje (%) de Operación con Retraso")
plt.xlabel("Aerolínea")
plt.ylim(0, max(stats_pd["porcentaje_retraso"]) + 10) # Espacio para las etiquetas
plt.grid(axis='y', linestyle='--', alpha=0.3)
plt.tight_layout()
plt.show()

# 1. Agrupar por Mes y calcular el promedio del retraso de salida
# Usamos el DataFrame 'df_delayed' para ver cuánto tiempo se retrasan cuando falla la puntualidad
retraso_mensual = df_delayed.groupBy("MONTH") \
    .avg("DEPARTURE_DELAY") \
    .orderBy("MONTH")

# 2. Convertir a Pandas para la visualización
retraso_mensual_pd = retraso_mensual.toPandas()

# Renombrar columnas para claridad
retraso_mensual_pd.columns = ['Mes', 'Promedio_Retraso']

# 3. Crear un mapeo de nombres de meses para que el gráfico sea más legible

# 4. Visualización con gráfico de líneas
plt.figure(figsize=(12, 6))

¿Qué tan frecuentes son los retrasos extremos?
 
Los retrasos extremos son súper raros en realidad. Solo pasan en el 0,83 % de los vuelos, o sea, unos 47.172 de miles que miramos.
La mayoría de los aviones llegan más o menos a la hora o incluso se adelantan un poco, y los que se pasan de las 3 horas de retraso son como esa “cola larga” de casos raros que se ven poco.
Por eso, aunque los retrasos gordos arman mucho ruido en las noticias y complican todo, la verdad es que la inmensa mayoría de los vuelos aterriza con muy poca diferencia respecto al horario previsto.
